# VIDHYATHI-PROJECT

INTRODUCTION 
 
 A Digital Clock Project :  In this  project the digital clock is made by using python programing language. It shows the current time in Hour :Minutes :Seconds format. 
 
 
A digital clock made by the using python programing which update every time . 
 
 
TOOLS USED : 
 
●	PYTHON (any version) 
●	TKINTER  
●	TIME MODULE 
 
 
FEATURES : 
 
●	In this project  it shows time in  Hours, minutes,second ● It  Update automatically. 
●	It easy to read by people like(students , children) 
 
 
 
HOW IT WORKS : 
 
●	Firstly we import all the libraries like - tkinter , strftime(for time) 
●	We also define the time_update because - by this it  update every second ● We use font styling and display the digital clock on the screen of the laptop. 
●	At last step we run the tkinter loop. 
■  Its shows  time.
